# Teste rápido

## Backend
1. `cd backend`
2. `npm install`
3. `npm run dev`
4. teste `GET /health`
5. crie cliente, pedido, item, motoboy e despacho

## Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
4. abra `http://127.0.0.1:5173/`

Na raiz do repo (opcional): `npm run dev:frontend` e `npm run dev:backend` após `npm install` em cada pasta.
